# 🚀 Gemini Hackathon Project

## Setup Instructions
1. **Clone the repo**
2. **Setup Backend:**
   - `cd backend`
   - `npm install`
   - Copy `.env.example` to a new file named `.env`
   - Paste your API Key in `.env`
   - Run `node server.js`
3. **Setup Frontend:**
   - `cd ../frontend`
   - `npm install`
   - `npm install ejs`
   - Run `npm start`

## Tech Stack
- **AI:** Gemini 2.5 Flash
- **Backend:** Node.js + Express
- **Frontend:** React + Axios
